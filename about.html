Honors Web Design

In this course, students learn the fundamentals of the internet, coding principles, HTML code and CSS code to create and maintain static websites using current industry tools and ememrging trends and technologies.

Advance Web Design

In this course, students continue applying their knowledge and skills of HTML and CSS and learn the JavaScript programming language, while learning to create interactive websites. Additionally, students begin a Structure Learning Experienc(SLE) by working with clients to build professional websites to specification

NJIT IS 2019 Advance Web Development

In this NJIT dual enrollment course, students build advaned websites with focus on sophisticated user interaction. Students continue participating in the Structured Learning Experience (SLE) by maintianing published websites for clients to specification

Technical Skills

Students will leave the program with a fluency in the following in-demand technical skills:
